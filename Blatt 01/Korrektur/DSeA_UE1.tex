\section*{Nr.1}

\subsection*{a)}
1.) Invariante:
Nach jedem Durchlauf ist jedes Element mit Index > newn richtig sortiert
\newline
\newline
2.) Schleife:
Nach jedem Schleifendurchlauf wird n mindestens um eins verkleinert bis 1
\newline


\subsection*{b)}
\underline{Worst-Case:} Liste in umgekehrter Reihenfolge
\newline
=> 1. Verschiebe erstes Element um n-1 Stellen
\newline
=> 2. Verschiebe zweites Element um n-2 Stellen
\newline
=> ....
\newline
Hieraus ergibt sich:
\newline 
\[\sum^{n-1}_{k=0}k = (n-1)\cdot \frac{n}{2} = \frac{n^{2}-n}{2}   \in O(n^{2})\]


\subsection*{c)}
- Große Elemente am Anfang werden früher nach hinten gebracht
\newline
- Kann um mehrere Plätze verschoben werden
\newline
- Kleines Element am Ende wird pro Durchlauf höchstens um 1 nach vorne verschoben

\subsection*{d)}
Da der Algorithmus eine ''größer'' und keine ''größergleich'' Operation durchführt, ist dieser stabil.

\section*{Nr.2}

\subsection*{a)}
\underline{I.A.:}    
\qquad{n = 0}
\[\sum^{0}_{k=1}2^{k}\cdot\left(k+1\right)=0\cdot2^{1}\]
\[\Leftrightarrow 0 = 0\]
\underline{I.B.:}    
\[\sum^{n}_{k=1}2^{k}\cdot\left(k+1\right)=n\cdot2^{n+1}\]
\underline{I.V.:}   
\newline 
\qquad{Die Behauptung gelte für ein festes n.}
\newline
\newline
\underline{I.S.:}   
\[\sum^{n+1}_{k=1}2^{k}\cdot\left(k+1\right)=\sum^{n}_{k=1}2^{k}\cdot\left(k+1\right) + 2^{n+1}(n+2)\]
\[= 2^{n+1}(n+n+2)\]
\[= 2\cdot2^{n+1}(n+1)\]
\[= 2^{n+2}(n+1)\]
\[q.e.d\]



\subsection*{b)}
\underline{I.A.:}    
\qquad{n = 0}
\[0=3\cdot0^{2}+0\]
\[\Leftrightarrow 0 = 0\]
\underline{I.B.:}    
\[\sum^{2n}_{k=n+1}2k=3n^{2}+n\]
\underline{I.V.:}   
\newline 
\qquad{Die Behauptung gelte für ein festes n.}
\newline
\newline
\underline{I.S.:}   
\[\sum^{2n+2}_{k=n+2}2k=\sum^{2n}_{k=n+1}2k-2(n+1)+2(2n+1)+2(2n+2)\]
\[=3n^{2}+n+2(3n+2)\]
\[=3(n^{2}+2n+1)+(n+1)\]
\[=3(n+1)^{2}+(n+1)\]
\[q.e.d\]


\subsection*{c)}
\underline{I.A.:}    
\qquad{Vermutung: m = 1, wähle n = 0}
\[0=0!-1\]
\[\Leftrightarrow 0 = 1-1\]
\[\Leftrightarrow 0 = 0\]
\underline{I.B.:}    
\[\sum^{n}_{k=1}k\cdot k!=(n+1)!-1\]
\underline{I.V.:}   
\newline 
\qquad{Die Behauptung gelte für ein festes n.}
\newline
\newline
\underline{I.S.:}   
\[\sum^{n+1}_{k=1}k\cdot k!=\sum^{n}_{k=1}k\cdot k!+(n+1)(n+1)!\]
\[= (n+1)! \cdot (1+n+1)-1\]
\[= (n+1)! \cdot (n+2)-1\]
\[= (n+2)!-1\]
\[q.e.d\]
